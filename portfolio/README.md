# 作品圖片使用說明

這個資料夾用於存放作品案例的圖片。

## 資料夾結構

- `chat-room/` - 即時聊天室系統的圖片
- `admin-system/` - 企業後台管理系統的圖片
- `student-project/` - 學生專題系統的圖片
- `deploy-platform/` - 自動化部署平台的圖片
- `ecommerce/` - 電商購物網站的圖片
- `data-dashboard/` - 資料分析儀表板的圖片

## 使用方式

1. 將你的作品圖片放入對應的資料夾中
2. 圖片命名建議：`1.jpg`, `2.jpg`, `3.jpg` 等
3. 支援的格式：JPG, PNG, GIF, WebP
4. 建議圖片尺寸：寬度至少 800px，保持 16:9 或 4:3 的比例

## 注意事項

- 如果資料夾中沒有圖片，系統會自動使用預設圖片
- 每個作品可以放置多張圖片，會以輪播方式顯示
- 圖片檔案大小建議控制在 500KB 以內，以獲得更好的載入速度
